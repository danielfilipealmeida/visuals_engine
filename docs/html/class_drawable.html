Detailed Description

base-class/interface for all classes that will be drawed on-screen

All classes that will be drawn should inherit from Drawable and implement all the virtual virtual methods

◆ draw() [2/2]

virtual void Drawable::draw ( ofRectangle _rect)
pure virtual

Draws the object.

The Draw function should be executed on every frame and will draw the content of the drawable object. This is executed on the main application draw() function. This draw method has the rectangle as a parameter and will use this instead of the internal this.rect.

Parameters
_rect- the rectangular area where to draw into
Note
This virtual method must be implemented on the class that inherits this class
