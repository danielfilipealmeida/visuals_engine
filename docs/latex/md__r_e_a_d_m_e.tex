{\bfseries{Visuals Engine}} is a library designed for creating real-\/time, reactive multimedia applications. Built with \href{https://openframeworks.cc/}{\texttt{open\+Frameworks}}, it provides a foundation for developers to craft dynamic visual experiences. While the repository includes example applications—such as a simple VJing tool—the primary focus is on the underlying engine tailored for new media and reactive multimedia projects.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Getting Started}{Getting Started}}\label{md__r_e_a_d_m_e_autotoc_md1}
