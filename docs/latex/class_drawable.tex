Draws the object.

This is executed on the main application {\ttfamily \doxylink{class_drawable_aa37d7b328240d343134adcfe5e4dcd38}{draw()}} function. This {\ttfamily draw} method has the rectangle as a parameter and will use this instead of the internal {\ttfamily this.\+rect}.
